# Credit Card UI

This is a simple credit card user interface fully designed and built by myself, with **HTML** and **CSS**.  
It is fully responsive and mobile-friendly.

## 🔧 Technologies Used

- HTML5
- CSS3
- Media Queries

## 💡 Features

- Clean and modern UI
- Responsive layout
- Custom chip and NFC icons
